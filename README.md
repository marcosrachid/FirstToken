## ERC-20 Ethereum Tokens

* ERC stands for Ethereum Request for Comments;
* ERC-20 standard specifies six functions which developers must use:
  * totalSupply
  * balanceOf
  * transfer
  * transferFrom
  * approve
  * allowance
* Widely accepted accross many cryptocurrency wallets.

## Premises

* Token name
